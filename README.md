# restify-router-error-sample
Sample repo to reproduce an issue with restify-router


Clone and run:

`yarn`

`yarn dev`

or

`npm install`

`npm run dev`


to start the server.

Routes are:

- /root
- /api-partners/another-route
- /api-partners/panel/a
- /api-partners/panel/b
- /api-partners/panel/c
